# Examples

Where end-to-end integration examples live. Empty until Phase 2 (when Sentinel
ships and there's something to integrate).

Planned examples:

- `nextjs-openai/` — Sentinel wrapping the OpenAI Node SDK in a Next.js app
- `flask-anthropic/` — Sentinel wrapping the Anthropic Python SDK in Flask
- `cloudflare-worker/` — Sentinel inside a Worker, telemetry to Guardian
- `langchain/` — Sentinel as a LangChain callback handler
