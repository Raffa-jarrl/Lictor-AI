Qué detecta Lictor

  • Clave service-role de Supabase expuesta en tu bundle JS
  • Tablas sin Row-Level Security (RLS) activa
  • Paquetes npm alucinados (un error típico de la IA)
  • Controles de autenticación solo en el frontend (puerta "admin" que no protege nada)
  • Webhooks de Stripe / GitHub / Shopify sin firma
  • Fugas de datos entre tenants por caché de Next.js
  • Endpoints de IA sin límite de tasa (rate limiting)
  • Hallazgos en lenguaje claro — sin jerga de SOC 2, nunca
