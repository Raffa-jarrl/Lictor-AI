O que o Lictor pega

  • Chave service-role do Supabase exposta no seu bundle JS
  • Tabelas sem Row-Level Security (RLS) ativo
  • Pacotes npm alucinados (erro clássico de IA)
  • Checagem de autenticação só no frontend ("portão admin" que não protege nada)
  • Webhooks de Stripe / GitHub / Shopify sem assinatura
  • Vazamento de dados entre tenants via cache do Next.js
  • Endpoints de IA sem rate limiting
  • Achados em português claro — sem jargão de SOC 2, nunca
