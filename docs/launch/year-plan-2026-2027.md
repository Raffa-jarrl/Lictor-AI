## How to use this doc month-by-month

Each month:

1. **First of the month:** Open this doc, read that month's section. Set the per-track shipping list as TODO items in your task system.
2. **Mid-month (15th):** Conductor agent produces a mid-month briefing. Compare actual progress vs the plan. Adjust scope if needed.
3. **End of month:** Mirror agent produces the monthly review. Compare metrics vs targets. Note risks materialized vs not. Hand-off to next month.
4. **End of quarter:** Run `/sprint-retro` (or equivalent) to compare the whole quarter's outcomes vs the plan. Adjust the next quarter's plan if reality has shifted materially.

The plan is a hypothesis. Reality will be different. The discipline is to *update the plan when reality teaches you something*, not to keep executing a plan reality has invalidated.
